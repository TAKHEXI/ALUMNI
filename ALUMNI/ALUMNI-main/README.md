# ALUMNI

ALUMNI一个开源的校友会平台

## 依赖

运行pip install -r requirement.txt安装以下依赖，或手动输入以下代码

```
django==3.2.5
xlrd==1.2.0
requests==2.25.1
requests-html==0.10.0
pyexcel-xls
```

## 目录介绍

后端项目的目录如下

```
Fduers/
|----Fuders/        存放django的配置文件
|----alumni/        存放后端的主要代码，包括数据库模型等一系列代码
|----functionality/ 存放功能性函数
|----generator/     存放测试数据生成器
```
